# Posture Patch photos

Add project images here, then reference them in `app/data/project-pages.js`:

```js
heroImage: '/images/posture-patch/hero.jpg',

images: [
  {
    src: '/images/posture-patch/device-front.jpg',
    alt: 'Front of the wearable',
    caption: 'Front view',
    layout: 'half', // or 'full'
  },
],
```

Suggested files:
- `hero.jpg` — main hero shot for the case study modal
- `device-front.jpg`, `device-side.jpg` — prototype photos
- `sketch.jpg` — early sketches or CAD renders
