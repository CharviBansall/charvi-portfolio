# Card images & logos

Put files here, then reference them in `app/data/cards.js`:

```js
image: { src: '/images/my-photo.jpg', alt: '...', position: 'background' }
logo: { src: '/images/my-logo.png', alt: '...', position: 'top-right', size: 'md' }
```

Supported formats: `.jpg`, `.png`, `.webp`, `.svg`, `.gif`
